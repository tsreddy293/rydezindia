# Rydez India

**Your Car. Your Income.**

India's trusted AI-powered vehicle sharing platform connecting verified vehicle owners with verified users.

## Tech Stack

- **Frontend:** Next.js 16, React 19, Tailwind CSS 4, Framer Motion
- **Backend:** Node.js, Express, MongoDB, JWT + OTP
- **Payments:** Razorpay (integration ready)
- **Maps:** Google Maps (integration ready)
- **AI:** OpenAI GPT (integration ready)

## Getting Started

### Frontend

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Backend API

```bash
cd server
npm install
cp .env.example .env
npm run dev
```

API runs on [http://localhost:5000](http://localhost:5000)
